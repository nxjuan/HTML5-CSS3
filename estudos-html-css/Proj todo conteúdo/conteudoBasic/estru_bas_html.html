Estrutura Basica de Um Documento Html

Divisão de Um Codigo HTML

1-Um codigo html pode ser dividido em algumas partes. A primeira parte, se inicia el !DOCTYPE html, isso quer dizer que o documento sera escrito em html, e tambem representa qual versão do html que será usada. No caso, quando se escreve so html, ao lado de !DOCTYPE, isso significa que a versão do html que sera usada é a mais recente, que é o HTML5.

2-Logo apos, é aberta a tag <html> junto com o atributo lang, que deve ser preenchido com a abreviação "pt-br". Isso indica que o documento html esta sendo escrito do português, e quando for aberto por uma pessoa estrangeira, dos estados unidos por exemplo, o site sera traduzido do portugues para o inglês atraves do translate do navegador.

3-Apos a abertura da tag html, temos a parte interna do programa, que é dividido em head e body.

4-A parte do head armazena as configurações do codigo html, assim como especificaçoes de comportamento e etc.

5-A tag body armazena as informações do codigo, dentro do body ficara o conteúdo observavel do codigo html.
