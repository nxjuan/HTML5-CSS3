HTML básico

hierarquia de titulo

A hierarquia de titulos, o nome é alto explicativo, é o conceito de tamanhos e importancia de titulos, onde o maior seria o h1, em seguida o h2, e assim sucessivamente.

caracteres

os caracteres em html são reservados para identificar as funções do proprio html, logo, se precisarmos usa-los dentro de um texto, ele causaria erro no codigo html. Por essa razão essistem os html entities, que são codigos que representam caracteres.

formatação de texto

as formatações de texto são as formas do html alterar as letras em função de alguma coisa que o programador nessecita representar. Como, dar enfase a uma palavra, identificar ela como nome cientifico, identificar aquela palavra ou trecho como importante, e até mesmo para identifica-lo como uma especie de codigo.

listas

as listas são divididas em duas subcategorias, as listas ordenadas e as listas não ordenadas. as ordenadas são listas que tem itens numerados, ja a outra possui apenas uma marcação de que algo é um item, contudo, não adiciona uma numeração.

CSS básico

cores

para representar cores no css, podemos usar os comandos background-color , que adiciona ao fundo uma cor de escolha, e background-image, que adiviona uma cor ao fundo, mas com o parametro linear-gradient() podemos representar varias cores em forma de degradê.

fontes

As fontes são uma parte muito importante de um site, para adicionar uma fonte basica, basta adicionar o codigo font-family, e inserir o nome da fonte desejada.
Contudo, se quisermos especificar que fonte queremos, ou se a fonte não existir no pacote basico do html, podemos importar seu link do google fontes(@importe url('---');...font-family: ;), ou adicionar uma fonte baixada:

font-family: ;

seletores personalizados

quando queremos atribuir estilo a uma tag, adicionamos a ela um seletor no css. contudo, este seletor vai estilizar todas as tags da mesma classe. para isso temos os seletores personalizados.
