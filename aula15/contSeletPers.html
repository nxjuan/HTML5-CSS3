SELETORES PERSONALIZADOS

Seletor "id"

O seletor id tem uma função muito especial. Dentro de um codigo css, quando queremos atribuir algum efeito a um codigo html, atribuimos um seletor css à tag desejada, contudo, a estilização será generalizada a todas as outras tags da mesma classe.

Para atribuir um id ao que desejamos esilizar, basta inserir o atributo id dentro da tag que desejada e então nomea-lo. Em seguida, inserimos um # seguido do nome do seletor, dentro do codigo css

NO HTML <h1 id="nome_do_id"> ////NO CSS #nome_do_id{ color= red;}

seletor "class"

o seletor class apresenta a mesma função do id, contudo, com uma vantagem, ele pode ser apricado a varias tags. Sem essa limitação do id, class aparenta ser muito mais vantajoso.

para usa-lo, basta inserir o atributo class dentro da tag desejada e então nomea-lo. Em seguida, inserimos um ".", seguido do nome que demos ao seletor, dentro do codigo css.

NO HTML <h1 class="nome_do_seletor"> ////NO CSS #nome_do_seletor{ color= red;}

OBS:!! Podemos adicionar as propriedades dos seletores não apenas às tags, mas tambem à frases, palavras e até mesmo a letras isoladas. Para isso, ultilizamos a tag span. Em seguida, apenas inserimos o atributo class à tag.

<p>Lorem, <span class="estilizar">ipsum dolor</span> sit amet consectetur</p>

OBS:!! O seletor id sobrepõe o seletor class, ou seja, se nessecitarmos de uma mudança extremamente pontual, podemos usa-lo sobre um id, para substituir a estilização inline.

OBS:!! Podemos inserir mais de uma classe em um mesmo atributo class (não precisamos separa-las por virgula):

<p class="estilizar1" class="estilizar2"> lorem ipsum <p>
